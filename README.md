# opensearch

Web search and page fetch for agents and TypeScript apps — one runtime, three packages.

```
@minpeter/opensearch        ← core runtime that routes providers and extracts pages
├── opensearch-mcp          ← exposes web_search / web_fetch as MCP stdio tools
└── opensearch-ai-sdk       ← exposes the same as Vercel AI SDK tools
```
